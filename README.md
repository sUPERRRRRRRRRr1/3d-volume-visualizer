# เครื่องมือหาปริมาตร 3 มิติ (Interactive 3D Volume Visualizer)

เครื่องมือเชิงโต้ตอบสำหรับนักเรียนระดับมัธยมปลาย/มหาวิทยาลัยปีต้น เพื่อ **เข้าใจว่าทำไมแผ่นบางๆ จึงรวมกันเป็นปริมาตร** ผ่านภาพ 3 มิติและแอนิเมชัน — ไม่ใช่แค่วาดกราฟสวยๆ UI ทั้งหมดเป็นภาษาไทย

รองรับ 2 โหมด:
- **ของแข็งจากการหมุน** (Solid of Revolution) — Disk/Washer & Shell
- **ปริมาตรจากภาคตัดขวาง** (Known Cross-Sections) — ฐานคือบริเวณใต้กราฟ/ระหว่างเส้น แล้วภาคตัดขวางตั้งฉากแกน X เป็นรูปสี่เหลี่ยมจัตุรัส/ครึ่งวงกลม/สามเหลี่ยมด้านเท่า/สามเหลี่ยมมุมฉาก โดย V = ∫A(x)dx

## ความสามารถ

- รับฟังก์ชันชัดแจ้ง `y = f(x)` เช่น `x^2`, `sin(x)`, `sqrt(x)`, `x^3 - 2*x` (ตรวจสอบและแจ้ง error เป็นภาษาไทย)
- หมุนรอบ **แกน X** หรือ **แกน Y**
- รองรับ **2 เส้นโค้ง** พร้อมหา **จุดตัดเชิงตัวเลข** อัตโนมัติ (sampling + bisection + ตรวจจุดสัมผัส) และแรเงาพื้นที่ระหว่างเส้น
- คำนวณปริมาตรด้วย **วิธีแผ่นจาน/วงแหวน (Disk/Washer)** และ **วิธีเปลือกทรงกระบอก (Shell)** — สลับได้บนของแข็งเดียวกันเพื่อแสดงว่าได้คำตอบเท่ากัน
- **Slider จำนวนแผ่น (n):** ดูค่าประมาณ (Riemann) ลู่เข้าสู่ค่าจริง
- **แอนิเมชันการหมุน:** กดปุ่มเพื่อดูพื้นที่ 2 มิติกวาดรอบแกนจนครบ 360°
- **ไฮไลต์แผ่นแบบโต้ตอบ:** เลือกตำแหน่ง x เพื่อดูรัศมีและพื้นที่หน้าตัดแบบเรียลไทม์
- **เฉลยทีละขั้น (KaTeX):** ถ้าเป็นพหุนามจะอินทิเกรตเชิงสัญลักษณ์จริงด้วยกฎยกกำลัง ได้ผลรูป `π·(เศษส่วน)`; ถ้าไม่ใช่พหุนามจะคำนวณเชิงตัวเลขด้วยกฎซิมป์สัน
- คำเตือนเมื่อเส้นโค้งข้ามแกนหมุน (ใช้ค่าสัมบูรณ์ของรัศมี)
- Responsive: เดสก์ท็อปเป็น 3 พาเนล มือถือเรียงแนวตั้ง

## วิธีรัน

```bash
npm install
npm run dev      # เปิด http://localhost:5173
npm run build    # สร้างเวอร์ชัน production ในโฟลเดอร์ dist/
```

## เทคโนโลยี

Vite · React 19 · Tailwind CSS v4 · Three.js + @react-three/fiber + drei · mathjs · KaTeX · zustand · ฟอนต์ Sarabun

## โครงสร้างหลัก

```
src/
├─ store/useAppStore.js        สถานะกลางทั้งหมด (zustand)
├─ hooks/useMathModel.js       รวมค่าที่คำนวณได้ (compile, จุดตัด, พื้นที่, ปริมาตร, เฉลย)
├─ lib/
│  ├─ mathParser.js            แปลง/ตรวจสอบสมการ (Problem 1)
│  ├─ intersection.js          หาจุดตัดเชิงตัวเลข (Problem 2)
│  ├─ numeric.js               simpson / trapezoid / bisection
│  ├─ volumeCalculator.js      ปริมาตรจริงตามแกน
│  ├─ symbolicIntegrator.js    อินทิเกรตพหุนามแบบเศษส่วน exact + เฉลย (Problem 4)
│  ├─ geometry.js              สร้าง mesh ของแข็งจากการหมุน + แผ่น disk/shell
│  ├─ crossSection.js          mesh ปริมาตรจากภาคตัดขวาง + crossSectionShapes.js
│  └─ colors.js, format.js
└─ components/                 ControlPanel · Plot2D · Viewport3D · SolutionPanel
```

## ขอบเขต (เวอร์ชันแรก)

รองรับเฉพาะ `y = f(x)`, หมุนรอบแกน X/Y, สูงสุด 2 เส้น — **ไม่** รองรับสมการ implicit, การหมุนรอบแกนใดๆ, หรือสมการ parametric

> หมายเหตุ: bundle ค่อนข้างใหญ่ (mathjs + three.js) ~620 kB gzipped ซึ่งปกติสำหรับแอปประเภทนี้ และทำงานฝั่ง client ทั้งหมด ไม่มี backend
